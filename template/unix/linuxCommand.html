File Related Commands

Commands Description
ls directory listing
ls -al formatted listing with hidden files
ls -lt Sorting the Formatted listing by time modification files
ls -l --color Lists files in “long” format As above, with colourized output
cd dir_Name change directory to dir_Name
cd change to home
pwd show current directory
mkdir dir_Name create a directory dir_Name
rm file_Name delete file_Name
rm -r dir_Name delete directory dir_Name
rm -f file_Name force remove file_Name
rm -rf dir_Name force remove directory dir_Name*
cp file_Name1 file_Name2 copy file_Name1 to file_Name2
cp -r dir_Name1 dir_Name2 copy dir_Name1 to dir_Name2; create dir_Name2 if it doesn't exist
mv file_Name1 file_Name2 rename or move file_Name1 to file_Name2 if file_Name2 is an existing dir_Nameectory, moves file_Name1 into dir_Nameectory file_Name2
ln -s file_Name link create symbolic link link to file_Name
touch file_Name create or update file_Name
cat file_Name Show contents of a file_Name
cat -n file_Name Show contents of a file, with line numbers
cat > file_Name places standard input into file_Name
cat file.a > file.b Copy content file.a to file.b
more file_Name output the contents of file_Name
head file_Name output the first 10 lines of file_Name
tail file_Name output the last 10 lines of file_Name
tail -f file_Name output the contents of file_Name as it grows, starting with the last 10 lines

File Permissions

Commands Description
chmod octal file Change the permission of file to octal,which can be found separately for user,group,world by adding,
• 4-read(r)
• 2-write(w)
• 1-execute(x)
chmod 777 read, write, execute for all
chmod 755 rwx for owner, rx for group and world For more options, see man chmod.

Process management

Commands Description

kill pid Kill the process with given pid
killall proc Kill all the process named proc
pkill pattern Will kill all processes matching the pattern
bg List stopped or background jobs,resume a stopped job in the background
fg Brings the most recent job to foreground
fg n Brings job n to the foreground

Compression

Commands Description

tar xf file.tar Extract the files from file.tar
tar czf file.tar.gz files Create a tar with Gzip compression
tar xzf file.tar.gz Extract a tar using Gzip
tar cjf file.tar.bz2 Create tar with Bzip2 compression
tar xjf file.tar.bz2 Extract a tar using Bzip2
gzip file Compresses file and renames it to file.gz
gzip -d file.gz gzip -d file.gz

Searching

Commands Description
grep pattern file Search for pattern in file
grep -r pattern dir Search recursively for pattern in dir
command | grep pattern Search pattern in the output of a command
locate file Find all instances of file
find . -name filename Searches in the current directory (represented by a period) and below it, for files and directories with names starting with filename
pgrep pattern Searches for all the named processes , that matches with the pattern and, by default, returns their ID
