What is Linux ?

Linux : Linux is the UNIX based open source operating system .
Just like Windows XP, Windows 7, Windows 8, and Mac OS X, Linux is an operating system.
An operating system is software that manages all of the hardware resources associated with your desktop or laptop.

What is Linux Kernel?

Linux Kernel : The kernel is the core of the system and manages the CPU, memory, and peripheral devices. The kernel serves as the connection between the hardware and software of a computer system, and it is the first program loaded during startup.
or The Linux Kernel is a low-level systems software whose main role is to manage hardware resources for the user. It is also used to provide an interface for user-level interaction.

What is open source?

Open source is Source code that anyone can inspect, modify, and enhance.
Open source is distribute software, including source codes freely to anyone who is interested. the user can add features and even debug and correct errors that are in the source code.

What is the difference between UNIX and LINUX ?

LINUX UNIX
Linux is an example of Open Source software development and Free Operating System (OS). Unix is an operating system that is very popular in universities, companies, big enterprises etc.
Linux is a Graphical User Interface with an optional Command Line Interface. Unix is a Command Line Interface.
BASH (Bourne Again SHell) is the Linux default shell. It can support multiple command interpreters. Originally the Bourne Shell. Now it's compatible with many others including BASH, Korn & C.
Linux OS is portable and can be executed in different Hard Drives. Unix is not portable.
In case of Linux, threat detection and solution is very fast, as Linux is mainly community driven and whenever any Linux user posts any kind of threat, several developers start working on it from different parts of the world Because of the proprietary nature of the original Unix, users have to wait for a while, to get the proper bug fixing patch. But these are not as common.
The Filesystems supported by Linux are as follows: xfs, ramfs, nfs, vfat, cramfsm ext3, ext4, ext2, ext1, ufs, autofs, devpts, ntfs The Filesystems supported by Unix are as follows: zfs, js, hfx, gps, xfs, gps, xfs, vxfs.
Examples of Linux are: Ubuntu, Fedora, Red Hat, Debian, Archlinux, Android etc. Examples of Unix are: OS X, Solaris, AIS, HP-UX, BSD, Iris, etc.
