Java String

String

String : String is a final class which is used to store and manage sequence of character.
or  String is an object that represents sequence of character values.

  • String class in java.lang package
  • String class implement Serializable and CharSequence interface.
  • String objects are immutable in java.
  • String and StringBuffer objects are thread safe.
  • StringBuilder objects are not thread safe.
  • toString(),hashCode() and equals() method is overrided.

Creating Strings

There are two ways to create String object:
  • By string literal
  • By new keyword

1.)    By string literal
  • String object is creating in string pull
  • string pull does not allows duplicate object
  • java is created new string object in string pull if there is no existing object or contained same or constant

2.)    By new keyword
  • Each time new key word is use a new string object is created in the heap memory

Example :

  1.)   String s1 = "apple" ;
      System.out.println(s1); // It print apple

  2.)   String s2 = new String("apple") ;
      System.out.println(s2); // It print apple

Immutable Object

Immutable : Immutable Object are those object whose state can not be changes.

To make type(class) immutable defined all the data member in final and initialize data member in a constructor.

Example 

public class Rectangle {
    final int length ;
    final int breadth ;

    Rectangle(int length , int breadth) {
        this.length = length ;
        this.breadth = breadth ;
    }

    @Override
    public String toString() {
        return "Rectangle [length = " + length + ",  breadth = " + breadth + "]";
    }
}

String is Immutable means

String is Immutable that means when string instance is created the same time instance can be change.
If we try to modify String , java create new object inside of modifing the existing object.

public class StringReverse {
    public static void main(String[] args) {
        String s = "java programs";
        System.out.println(reverse(s));
        System.out.println(reverseWithRecartion("Hello")); // method call recurcation function
//*****  reverse with built in method call  **//
        StringBuilder input = new StringBuilder("how are you");
        System.out.println(input.reverse());
    }
//*****  reverse with simple way ***//
    public static String reverse(String s){
        StringBuilder sb = new StringBuilder();
        for (int i = s.length() - 1 ; i >= 0 ; i--) {
            sb.append(s.charAt(i));
        }
        return sb.toString() ;
    }
//***** reverse with recurcation function   ***//
    public static String reverseWithRecartion(String s){
        if((s == null) || (s.length() <= 1)){
            return s;
        }
        return reverseWithRecartion(s.substring(1)) + s.charAt(0) ;
    }
}

    	s = s.toLowerCase();
