Java Constructor

Introduction

Constructor : Constructor is a special type of method which execute each time when an Object is created with the help of new key word. Constructor is used to initialize the Object.

  • Constructor is always same as its class name
  • There is no return type of Constructor
  • static modifiers is not allow for Constructor
  • There are two type of Constructor 1. Defaul Constructor 2. Paramiterize Constructor
  • If in class there is no Constructor then java implicitly (automatically) add default Constructor
  • In one class we can create more Constructor but every Constructor should different type of parameters or number of parameters
Constructor Overloading : Multiple Constructor in a class with differ in parameter lists is known as Constructor Overloading.

Constructor Overloading 

public class Student {
    int id;
    String name;
    int age;
    Student(int i, String n) {
        id = i;
        name = n;
    }
    Student(int i, String n, int a) {
        id = i;
        name = n;
        age = a;
    }
    void display() {
        System.out.println(id + " " + name + " " + age);
    }

    public static void main(String args[]) {
        Student s1 = new Student(100, "Anup");
        Student s2 = new Student(101, "Kamal", 20);
        s1.display();
        s2.display();
    }
}

Copy Constructor 

public class Employee {
    int id;
    String name;
    Employee(int i,String n){
    id = i;
    name = n;
    }
    Employee(Employee e){
        id = e.id;
        name =e.name;
    }
    void display(){
        System.out.println(id+" "+name);
    }

    public static void main(String args[]){
    Employee e1 = new Employee(111,"Rahul");
    Employee e2 = new Employee(e1);
    e1.display();
    e2.display();
   }
}

If we place return type in constructor What will happen ?

It will not given any compile or run time error , because compiler and JVM considers it as a method.

Can we declare constructor as private?

Yes we can declare constructor as private. All four access modifiers are allowed to constructor. we declare private constructor for Singleton class.

When compiler provides default constructor?

If there is no constructor defined by developer then compiler provides default constructor . compiler provide default constructor only if there is no explicit constructor.
